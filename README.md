# spinacia-cli
react cli

## Install

``` bash
npx spinacia-cli
```
or

``` bash
npx spinacia-cli <folder name>
```

*[npx](https://medium.com/@maybekatz/introducing-npx-an-npm-package-runner-55f7d4bd282b) comes with npm 5.2+ and higher, run ```npm i -g npm``` to update
